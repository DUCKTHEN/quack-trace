# Roadmap

Quack Trace is currently beta. This list tracks known improvements and experiments.

## Before Wider Sharing

- Improve mobile layout so the tool remains usable on narrow screens.
- Recheck GitHub Pages behavior on desktop and mobile.
- Add a clearer beta notice in the UI if needed.
- Confirm exported sample files open in Illustrator, Blender, CLO, and Marvelous Designer as expected.

## Editing Workflow

- Add points into an existing closed shape or line.
- Support moving selected lines with arrow keys.
- Support moving visible curve handles with arrow keys in more modes.
- Make Delete key behavior predictable for points, lines, and closed shapes.
- Prevent broken geometry after deleting points from a closed shape.

## Grid And Scale

- Improve grid snap wording and feedback.
- Consider selectable snap intervals, such as 5 mm, 1 cm, and 5 cm.
- Keep grid paper, canvas grid, and scale square visually consistent.

## Export Experiments

- Keep SVG as the reliable curve-preserving format for Illustrator.
- Treat DXF, MD/CLO Python, and Blender Python as experimental exports.
- Document known limitations for each export target.

## Nice To Have

- Better temporary save / restore workflow.
- More compact controls for small screens.
- Optional release sample files.
