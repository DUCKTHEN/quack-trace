# Privacy

Quack Trace is a static browser app.

## What Stays Local

- Images loaded through the Image button stay in your browser session.
- Traced points, curves, faces, and scale settings stay in browser memory unless you export them.
- JSON, SVG, CSV, DXF, and Python files are generated locally by your browser.

## What Is Hosted

GitHub Pages hosts only the app files:

- HTML
- CSS
- JavaScript
- App icon
- UI sound files
- Documentation files

## What Is Not Uploaded By The App

Quack Trace does not intentionally upload:

- Your images
- Your pattern data
- Your exported files
- Your local file paths

## Notes

If you publish or share an exported file yourself, that exported file may contain the coordinates and names you created in Quack Trace.
